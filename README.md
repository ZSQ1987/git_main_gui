# Git Main GUI 工具

一个功能强大的Git命令可视化工具，帮助开发者更方便地执行Git命令，提高工作效率。

## 功能特点

- **双栏布局**：左侧显示工作目录、命令输入和执行结果，右侧显示常用Git命令
- **常用命令快捷按钮**：分为创建相关和更新相关命令，点击即可填充到输入框
- **手动添加命令**：支持用户自定义添加常用命令
- **工作目录选择**：可切换Git命令的执行目录
- **命令执行输出**：实时显示命令执行结果，区分正常信息和错误信息
- **功能说明**：点击功能说明按钮查看详细使用指南
- **窗口居中**：启动时窗口自动居中显示
- **支持ssh-keygen**：自动处理交互式命令，避免程序死机

## 技术实现

- 使用Python 3.11.9 + tkinter/ttk开发
- 支持Windows系统
- 自动检测Git Bash路径
- 非交互式执行ssh-keygen命令

## 使用方法

1. 运行 `python git_main_gui.py` 启动应用程序
2. 在左侧工作目录区域选择Git仓库目录
3. 在右侧点击常用命令按钮，或在左侧输入框中手动输入Git命令
4. 点击"执行命令"按钮执行命令
5. 在下方输出区域查看命令执行结果
6. 点击"清空输出"按钮清空输出内容
7. 点击"功能说明"按钮查看详细使用指南

## 常用命令

### 创建相关命令
- 初始化仓库 (git init)
- 克隆仓库 (git clone)
- 创建分支 (git checkout -b)
- 添加远程仓库 (git remote add)
- 创建README.md
- 创建Python .gitignore
- 创建MIT License

### 更新相关命令
- 查看状态 (git status)
- 添加所有文件 (git add .)
- 提交（需补信息）
- 推送到远程 (git push)
- 拉取远程代码 (git pull)
- 查看分支 (git branch)
- 切换分支 (git checkout)
- 查看远程仓库 (git remote -v)
- 首次推送 (git push -u)
- 上传main分支

## 注意事项

- 执行命令前请确保已安装Git，并配置好环境变量
- 执行git commit时需手动补充提交信息（如 git commit -m "更新内容"）
- 输出区红色文字为错误信息，黑色为正常执行结果
- 若未找到Git Bash，请按照提示安装Git并配置环境变量
